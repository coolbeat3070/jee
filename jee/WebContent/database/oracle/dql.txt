검색
select column [(as) "별칭"], ... from table;
where column조건
-select 문: 제한된 column을 추출(데이터만을 선정해서 추출할 수 없다.)
-where 옵션 :조건절 (제한된 행을 추출) : 참인 행을 추출
-실행순서 : from -> where -> select
-별칭사용 : space bar을 사용한다.
-select (select..),column이 가능하다. 
단, 서브쿼리에서의 추출 데이터는 하나만 있어야 한다. 
여러개의 데이터가 추출될 경우는 오류로 column를 ','없이 적용하여 
문법적 오류가 발생하기 때문이다.
